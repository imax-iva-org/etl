Назначение:

Получение и загрузка данных с использованием Facebook Graph API.


Файлы проекта:

fb.py		- основной файл проекта (скрипт)
config.ini	- конфигурационные параметры, необходимые для работы скрипта
sql_tmplt.sql	- файл-шаблон, использующийся для загрузки данных
etl.log		- лог с ошибками


Дополнительные файлы:

init.sql
Используется для инициализации объектов СУБД. Выполняется пользователем с правами администратора.

etl.bat
Пакетный файл для запуска скрипта из планировщика задач. Содержит команды активации и деактивации виртуального окружения проекта.

requirements.txt
Список используемых пакетов.


Замечания:

В конфигурационном некоторые значения параметров изменены на символы "*", т.к. проект находится в публичном доступе (client_ID, client_secret, access_token).
В качестве СУБД используется MySQL Server 5.7.
